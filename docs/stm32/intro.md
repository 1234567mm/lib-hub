# STM32知识库

STM32是STMicroelectronics（意法半导体）推出的32位微控制器系列，基于ARM Cortex-M内核，广泛应用于工业控制、消费电子、物联网等领域。

## 栏目结构

```
STM32知识库
├── 基础知识
│   └── GPIO、基本定时器、时钟配置等
├── 外设驱动
│   └── USART、SPI、I2C、ADC、PWM等
└── 项目实战
    └── 完整项目案例与分析
```
