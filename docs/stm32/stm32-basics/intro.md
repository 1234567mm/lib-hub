# STM32基础知识

本栏目聚焦STM32开发中的核心基础知识，涵盖寄存器配置、HAL库使用、时钟系统等入门必学内容。

## 内容列表

- [GPIO端口配置](./gpio-config)
- [时钟系统详解](./clock-system)
- [中断基础入门](./interrupt-basics)
- [定时器基本原理](./timer-basics)

## 推荐学习顺序

1. **GPIO配置** - 学会控制LED、按键
2. **时钟系统** - 理解时钟树与配置
3. **中断基础** - 掌握外部中断
4. **定时器** - 学习定时计数功能

---

*基础知识是嵌入式开发的根基，建议扎实掌握后再进行外设学习。*
