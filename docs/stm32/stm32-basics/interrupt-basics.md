---
title: 中断基础
---

概述 STM32 中断模型、优先级与常见使用模式，以及如何在应用中安全地使用中断。

## 中断模型
- Cortex-M 内核提供 NVIC，用于管理外设中断的优先级与使能。

## 常见步骤
1. 配置外设中断源（例如使能 USART 中断、定时器中断等）。
2. 在 NVIC 中设置优先级并使能对应 IRQn。
3. 在中断服务例程（ISR）中尽量缩短执行时间，避免阻塞操作。

## 示例（伪代码）
```c
void TIM2_IRQHandler(void) {
	if (timer_update_flag) {
		// 快速处理
		timer_clear_flag();
	}
}
```

## 最佳实践
- 将复杂或耗时的处理推到主循环或任务（通过事件/消息传递）。
- 使用适当的优先级策略，避免优先级反转。
