这是关于 STM32 GPIO（通用输入输出）配置的分步指南，包含常见用例、注意事项和示例代码。

## 概要
GPIO 用于将 MCU 引脚配置为输入、推挽输出、开漏输出或复用外设功能。不同 STM32 系列寄存器略有不同，请参考具体芯片手册。

## 前置条件
- 熟悉 STM32Cube 或寄存器编程基础
- 拥有目标芯片的数据手册（RM）和参考手册

## 基本配置步骤（概念）
1. 使能对应端口时钟（RCC）。
2. 设置引脚模式（输入/输出/复用/模拟）。
3. 配置输出类型（推挽/开漏）和速率。
4. 可选：配置上拉/下拉电阻。

## 简单示例（伪代码）
```c
// 伪代码，仅说明寄存器顺序
RCC_EnableGPIOAClock();
// 将 PA0 设为通用推挽输出
GPIOA->MODER &= ~(3 << (0 * 2));
GPIOA->MODER |=  (1 << (0 * 2));
GPIOA->OTYPER &= ~(1 << 0); // 推挽
GPIOA->OSPEEDR |= (1 << (0 * 2));
```

## 注意事项
- 使用 HAL/LL 时，优先使用库接口以提高可移植性。
- 更改 GPIO 前请确认时钟已正确开启。
