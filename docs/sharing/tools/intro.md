# 工具推荐

好的工具能让开发效率事半功倍。这里推荐嵌入式开发中实用的工具软件，涵盖IDE、调试工具、辅助软件等。

## 开发环境

| 工具 | 特点 | 适用场景 |
|------|------|----------|
| **STM32CubeIDE** | ST官方免费IDE，基于Eclipse | STM32开发首选 |
| **Keil MDK** | 行业最广泛，仿真强大 | ARM开发经典 |
| **IAR EWARM** | 编译效率高，代码紧凑 | 商业项目 |
| **VS Code + PlatformIO** | 轻量级，插件丰富 | 快速原型开发 |

## 调试工具

| 工具 | 特点 | 适用场景 |
|------|------|----------|
| **ST-Link** | ST官方调试器，价格实惠 | STM32调试 |
| **J-Link** | 高速、兼容性好 | 高端ARM调试 |
| **逻辑分析仪** | 24MHz采样，协议分析 | 串口/SPI/I2C调试 |
| **示波器** | 信号观察、时序分析 | 硬件调试 |

## 辅助软件

- **Serial Plot** — 串口数据可视化
- **Altium Designer** — PCB设计
- **Fritzing** — 快速原型绘制
- **draw.io** — 架构图绘制

## 效率工具

- **snipaste** — 截图 + 贴图 + 取色
- **Listary** — 全局搜索 + 文件定位
- **Notion** — 笔记与知识管理
